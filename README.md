# Crud-Angular

# Como usar:

### Backend:
<p> Abra o projeto do backend </p>
<p> Use o `npm i` para instalar as dependencias.  </p>
<p> Rode o `npm start` para rodar o servidor.  </p>

### Frontend:
<p> Abra o projeto frontend </p>
<p> Use o `npm i` para instalar as dependencias. </p>
<p> Rode o  `ng serve --o` para rodar </p>

### O que foi usado:
> Angular,

> Material Desing,

> JsonServer,

> JavaScript,
